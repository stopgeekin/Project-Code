# Project-Code
A full-stack web application integrating front-end interactivity with back-end functionality using C#
Architected a normalized SQL schema to manage many-to-many relationships between titles, authors, and transactions. This mirrors the logic required to manage player tracking data and scouting reports.
Built robust backend logic to "clean and validate ad-hoc datasets" , ensuring that inventory and user data remain accurate and "well-tested".
Leveraged modular C# logic and reusable JavaScript components to ensure a "scalable" system, following modern software engineering best practices.
Developed an intuitive UI/UX with a "responsive layout" using HTML and CSS, similar to the "dashboards and data visualizations"  used by coaching and front-office staff.
